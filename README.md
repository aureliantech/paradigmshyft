# ParadigmSHYFT Website

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Leave all settings as default → Deploy

## Adding Team Photos

Replace the placeholder images with real photos:

- `adam.jpg` — Replace with Adam's photo (CEO)
- `shannon.jpg` — Replace with Shannon's photo (Operations Assistant)

Photos work best at a 4:5 portrait ratio (e.g. 600×750px).
